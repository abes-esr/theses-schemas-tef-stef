Commentaire

Le lien vers les données d'autorité est obligatoire. Doit donc posséder soit autoriteExterne , soit autoriteInterne , mais pas les deux.

Règles d'écriture
Voir les éléments enfants
UNIMARC
Zone 702 $a pour le nom, $b pour le prénom, $4 727 pour le code de fonction.
Zone 200 $g en rétablissant l'ordre : [Prénom Nom], sans virgule entre les deux éléments du nom. Si plusieurs occurrences, les différents noms doivent figurer dans le même $g, séparés par une virgule et un espace.
Exemple(s)
Houssel, Jean-Pierre (1934-….) 
 <marc.thesisAdvisor>
  <name>Houssel, Jean-Pierre</name>
  <autoriteExterne autoriteSource="Sudoc">9026925508</autoriteExterne>
 </marc.thesisAdvisor>
