Commentaire
Cet élément peut contenir une description DC classique (titre, auteur...). Il peut aussi se contenter de pointer vers un URI identifiant la ressource (avec dc:identifier). Enfin, il peut contenir une citation bibliographique non structurée de la ressource (avec dcterms:bibliographicCitation).
Pour décrire chaque ressource externe, il faut répéter le bloc mets:dmdSec de type "tef_desc_externe".
